# Tienda de Consignación - Aplicación de Consola en .NET Core

![.NET Core](https://img.shields.io/badge/.NET-Core-512BD4?style=flat) ![C#](https://img.shields.io/badge/C%23-Language-239120?style=flat) ![License](https://img.shields.io/badge/License-MIT-blue.svg)

## Descripción

**Tienda de Consignación** es una aplicación de consola desarrollada en **C#** usando **.NET Core** que permite registrar y visualizar productos disponibles para la venta en una tienda de consignación.  
El usuario puede interactuar con la aplicación para:

- Visualizar los artículos disponibles.
- Comprar un producto ingresando su ID.
- Actualizar automáticamente el estado del artículo vendido.
- Evitar compras duplicadas de productos ya vendidos.

Esta aplicación tiene fines **académicos y didácticos**, y hace uso de **listas de tuplas** para almacenar los productos.

---

## Características

- Lista inicial de productos con ID, título, descripción, precio y estado de venta.
- Interfaz de consola sencilla y clara.
- Validación de entradas del usuario (ID numérico y comandos de salida).
- Actualización dinámica del estado de los artículos al realizar una compra.
- Mensajes de confirmación y advertencia para compras inválidas o artículos ya vendidos.

## Requisitos

- **Visual Studio 2022** o superior / **Visual Studio Code**
- **.NET Core SDK 6.0** o superior
